## How to install 

```bash
  # Clone the project
  git clone https://github.com/whosramoss/react-money/

  # Go to the project directory
  cd react-money

  # Install dependencies
  npm install
```

## How to run 
```bash
  # Start the app 
  npm run dev

  # Start the server 
  npm run dev:server
```

## How to build 
- Run the command:
```bash
  npm run build
```
